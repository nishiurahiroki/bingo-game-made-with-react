## develop
### must be installed
* **nodejs**  
(>= 8.0.x)
* **npm**  
(>= 5.0.x)

### set up node.js
```sh
nvm install
```

### set up
```sh
npm i
```

### build
```sh
# build
npm run build:develop

# build & watch
npm run build:watch:develop
```

### deploy
```sh
npm run deploy # use firebase
```

### endpoint
```
./index.html
```
